# 1. Pengenalan Exploratory Data Analysis (EDA)

### **Apa itu EDA?**
Exploratory Data Analysis (EDA) adalah proses mempelajari struktur data melalui teknik statistik dan visualisasi guna menemukan pola, anomali, hubungan antar variabel, dan wawasan awal sebelum melakukan modeling.

### **Fungsi Utama EDA:**
- Memahami karakteristik data  
- Menemukan nilai ekstrim (outlier)  
- Menilai distribusi data  
- Mengidentifikasi korelasi antar variabel  
- Menentukan kebutuhan pembersihan dan transformasi data  

### **Tahapan Umum:**

| Tahap                    | Kegiatan Utama                                 |
|--------------------------|-------------------------------------------------|
| Memahami Struktur Data   | Melihat info, ringkasan statistik, tipe data    |
| Univariate Analysis      | Analisis satu variabel: distribusi, outlier     |
| Bivariate Analysis       | Analisis dua variabel: korelasi, hubungan       |
| Multivariate Analysis    | Analisis beberapa variabel sekaligus            |
| Visualisasi Data         | Histogram, boxplot, scatter, heatmap, dll       |
